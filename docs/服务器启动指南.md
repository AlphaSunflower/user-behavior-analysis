# 服务器启动指南

> 服务器 IP：192.168.100.140  
> 操作系统：Linux（CentOS 7+）  
> 适用场景：首次部署或重启服务器后需要按顺序启动的服务

---

## 服务启动顺序

```
1. MySQL      (3306)
2. Hadoop     (HDFS + YARN)
3. Kafka      (9092)
4. Spark      (提交 Streaming 作业)
5. Nginx      (80)
```

---

## 一、MySQL 5.7.44

```bash
systemctl start mysqld
# 或
service mysqld start

# 验证
mysql -u root -p123456 -e "SELECT 1"
```

---

## 二、Hadoop 3.3.5

```bash
# 启动 HDFS
start-dfs.sh

# 启动 YARN（Spark 需要）
start-yarn.sh

# 验证
jps
# 应看到：NameNode, DataNode, ResourceManager, NodeManager
hdfs dfs -ls /
```

---

## 三、Kafka 2.6.0

```bash
# Kafka 依赖 ZooKeeper（如未单独安装，使用 Kafka 自带的）
# 如果 ZooKeeper 未启动：
/home/hadoop/opt/app/kafka/bin/zookeeper-server-start.sh -daemon /home/hadoop/opt/app/kafka/config/zookeeper.properties

# 启动 Kafka
/home/hadoop/opt/app/kafka/bin/kafka-server-start.sh -daemon /home/hadoop/opt/app/kafka/config/server.properties

# 验证
/home/hadoop/opt/app/kafka/bin/kafka-topics.sh --list --bootstrap-server 192.168.100.140:9092
# 应看到：user_behavior, user_blacklist, __consumer_offsets
```

### Kafka 关键配置确认

```bash
grep -E "listeners|advertised" /home/hadoop/opt/app/kafka/config/server.properties
# 应输出：
# listeners=PLAINTEXT://0.0.0.0:9092
# advertised.listeners=PLAINTEXT://192.168.100.140:9092
```

## 四、Spark Streaming（提交到 YARN）

```bash
# 将本地打包的 JAR 上传到服务器
scp kafka-spark/target/scala-2.12/UserBehaviorAnalysis-assembly-1.0.jar root@192.168.100.140:/home/hadoop/

# 提交 Spark 作业
spark-submit \
  --class streaming.UserBehaviorStreaming \
  --master yarn \
  --deploy-mode cluster \
  --driver-memory 1g \
  --executor-memory 1g \
  --executor-cores 1 \
  /home/hadoop/UserBehaviorAnalysis-assembly-1.0.jar

# 查看作业状态
yarn application -list
```

## 五、Nginx 1.26.1

```bash
# 将前端构建产物上传到服务器
scp -r frontend/dist/* root@192.168.100.140:/usr/share/nginx/html/

# 启动 Nginx
systemctl start nginx
# 或
nginx

# 验证
curl http://localhost
```

### Nginx 配置参考

```nginx
server {
    listen 80;
    server_name _;

    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 一键启动脚本（服务器端）

```bash
#!/bin/bash
# 保存为 /home/hadoop/start-all-services.sh

echo "=== 启动 MySQL ==="
systemctl start mysqld

echo "=== 启动 Hadoop ==="
start-dfs.sh
start-yarn.sh

echo "=== 启动 Kafka ==="
/home/hadoop/opt/app/kafka/bin/zookeeper-server-start.sh -daemon /home/hadoop/opt/app/kafka/config/zookeeper.properties
sleep 5
/home/hadoop/opt/app/kafka/bin/kafka-server-start.sh -daemon /home/hadoop/opt/app/kafka/config/server.properties
sleep 5

echo "=== 验证 Kafka ==="
/home/hadoop/opt/app/kafka/bin/kafka-topics.sh --list --bootstrap-server 192.168.100.140:9092

echo "=== 启动 Nginx ==="
systemctl start nginx

echo "=== 全部服务启动完成 ==="
```
